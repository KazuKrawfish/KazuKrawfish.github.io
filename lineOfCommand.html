Line of Command

Line of Command is a turn based strategy game set in the violent world of Redonia. The Redonian army faces numerous enemies and must defeat them all to secure the peace for its hard-working citizenry.

The game uses a custom set of sprites, and the interface operates within the Simple Fast Media Library (SFML) API.

The computer player makes (relatively) intelligent combat and production decisions, including decent path-finding, as well as selecting the best units to attack the enemies it encounters.
